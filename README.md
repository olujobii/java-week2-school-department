# A SCHOOL MANAGEMENT APPLICATION
## DESCRIPTION
A School management command-line application that demonstrates Object-Oriented Programming and its principles

NOTE: Admin access password is 12345

**Features**
- List total number of courses, students and staffs
- Registering an applicant into school database
- Approving an applicant to be enrolled as a student through admin priviledges
- Expel a student through admin priviledges
- Loose coupling of code with the use of interface
